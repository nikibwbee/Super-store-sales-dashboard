# Super Store Sales Dashboard

## Overview
This project is an interactive Power BI dashboard developed using the Super Store Sales dataset. The dashboard provides insights into sales performance, profit trends, customer segments, regional performance, and sales forecasting to support data-driven decision-making.

## Objectives
- Analyze overall sales and profit performance
- Identify top-performing states and regions
- Compare sales across customer segments and categories
- Monitor monthly sales and profit trends
- Generate sales forecasts for future planning

## Tools & Technologies
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

## Dashboard Features

### KPI Metrics
- Total Sales
- Total Profit
- Total Orders
- Average Shipping Days

### Visualizations
- Sales by State
- Sales by Region
- Sales by Segment
- Sales by Category
- Sales by Payment Mode
- Monthly Sales Trend
- Monthly Profit Trend
- Sales Forecasting

### Interactive Features
- Region Filter
- Category Filter
- Date Slicer
- Cross-Filtering Between Visuals

## Key Insights
- Identified the highest-performing states based on sales revenue.
- Analyzed customer purchasing behavior across different segments.
- Compared regional sales performance.
- Evaluated sales trends over time.
- Generated sales forecasts to support business planning.

## Dataset
The dataset contains information related to:
- Orders
- Customers
- Products
- Sales
- Profit
- Shipping Details
